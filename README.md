# 图片批量下载程序

这是一个功能强大的图片批量下载工具，支持从网页中自动提取和下载图片，具有以下特性：

## 功能特性

- 🌐 **智能网页解析**: 自动解析网页内容，提取图片链接和子页面链接
- 📁 **自动目录管理**: 根据网页链接自动创建对应的目录结构
- 🚀 **多线程下载**: 支持多线程并发下载，提高下载效率
- 📊 **实时进度显示**: 直观的进度条和状态显示
- 🎯 **智能过滤**: 支持按图片类型和大小过滤
- 📅 **任务调度**: 支持计划任务和定时下载
- 🔍 **通配符支持**: 支持使用通配符创建批量下载任务
- 🎨 **友好界面**: 简洁直观的图形用户界面

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行主程序：
```bash
python main.py
```

2. 在界面中输入要下载的网页URL
3. 设置下载参数（目录、过滤条件等）
4. 点击开始下载

## 项目结构

```
xiuren/
├── main.py                 # 主程序入口
├── requirements.txt        # 依赖包列表
├── config.json            # 配置文件
├── src/                   # 源码目录
│   ├── core/              # 核心功能模块
│   ├── gui/               # 图形界面模块
│   ├── utils/             # 工具模块
│   └── models/            # 数据模型
├── logs/                  # 日志文件
└── downloads/             # 默认下载目录
```

## 配置说明

程序使用 `config.json` 文件进行配置，主要参数包括：

- `max_threads`: 最大下载线程数
- `timeout`: 请求超时时间
- `retry_times`: 重试次数
- `download_path`: 默认下载路径
- `image_filters`: 图片过滤规则
