# 图片批量下载程序

这是一个面向网页图集的图片批量下载工具，支持从页面中自动提取图片链接、创建下载任务、并发下载、过滤异常文件名，并在图形界面中展示实时进度。项目当前包含核心下载器、网页解析器、批量任务创建、并发与性能优化、安全校验、统一异常处理、配置管理和 GUI 进度窗口等模块。

## 功能特性

- 🌐 **智能网页解析**: 自动解析网页内容，提取图片链接和子页面链接
- 📁 **自动目录管理**: 根据网页链接自动创建对应的目录结构
- 🚀 **多线程下载**: 支持多线程并发下载，提高下载效率
- 📊 **实时进度显示**: 直观的进度条和状态显示
- 🎯 **智能过滤**: 支持按图片类型和大小过滤
- 📅 **任务调度**: 支持计划任务和定时下载
- 🔍 **通配符支持**: 支持使用通配符创建批量下载任务
- 🎨 **友好界面**: 简洁直观的图形用户界面

## 安装依赖

```bash
pip install -r requirements.txt
```

如果需要运行测试或开发调试，请安装开发依赖：

```bash
pip install -r requirements-dev.txt
python run_tests.py
```

## 使用方法

1. 运行主程序：

```bash
python main.py
```

2. 在界面中输入要下载的网页URL
3. 设置下载参数（目录、过滤条件等）
4. 点击开始下载

## 适用场景

- 批量抓取网页中的图片资源，并按页面或任务自动归档。
- 对图集页面、分页页面、子页面链接进行解析和批量任务生成。
- 在下载时过滤无效文件名、异常 URL、过小图片或不符合规则的资源。
- 使用多线程、退避重试、限速和服务端友好配置降低失败率。
- 通过 GUI 查看任务进度，或通过脚本模块集成到自动化流程中。

## 版本控制约定

- `downloads/` 下除示例目录外的真实下载内容不提交。
- `logs/`、`__pycache__/`、`.pytest_cache/`、覆盖率文件和本地工具状态不提交。
- 配置、源码、测试、文档和少量可说明项目行为的示例数据可以提交。
- 如果要分享完整下载结果，建议使用压缩包、对象存储或 Release 附件，而不是直接提交到主分支。

## 配置说明

程序使用 `config.json` 文件进行配置，主要参数包括：

- `max_threads`: 最大下载线程数
- `timeout`: 请求超时时间
- `retry_times`: 重试次数
- `download_path`: 默认下载路径
- `image_filters`: 图片过滤规则
